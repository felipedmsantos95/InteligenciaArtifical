-- Mutagenesis example: CILP++ --

This zip folder contains two subfolders: mutaFull and mutaBond

* mutaFull is a CILP++ learning example with the mutagenesis dataset containing the entirety of the background knowledge, 
including expert knowledge on well-known molecular structures.

* mutaBond is also a mutagenesis example, but containing only bonds and atoms descriptions

-> To see the already obtained results on both examples, please check files/main.log.
-> To re-run the experiments, please double-click cilp.exe.
